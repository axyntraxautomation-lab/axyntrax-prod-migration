## Arquitectura del Ecosistema IA
1. **`/jarvis` (CEO Agent)**: Orquestación maestra, sincronización con Google Workspace y Gmail Maestro (Gestión inteligente de leads entrantes).
2. **`/cecilia` (Commercial Agent)**: Recepcionista virtual, análisis NLP de necesidades del cliente, y motor de presupuestación inteligente con márgenes blindados (40%).
3. **`/neo` (DevOps Agent)**: Automatización de ciclo de vida de proyectos. Traduce presupuestos a microtareas, ejecuta scaffolding de código y simulación de CI/CD.
4. **`/atlas` (Guardian Agent)**: Vigilancia de salud del sistema (99.9% Uptime). Detecta caídas y ejecuta rutinas de autorreparación instantánea.
